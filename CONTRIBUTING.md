# Contributing to ASP.NET Core Bookstore

First off, thank you for considering contributing to this repository! It's people like you that make the open-source community such an amazing place to learn, inspire, and create.

## Getting Started

1. Fork the repository and create your branch from `master`.
2. Ensure you have the .NET SDK installed.
3. If you've added code that should be tested, add tests!
4. Ensure the test suite passes (`dotnet build` & `dotnet test`).
5. Update the documentation (README, etc.) if applicable.

## Pull Request Process

1. Keep your Pull Requests small and focused on a single feature or bug fix.
2. Link your Pull Request to an existing issue if one exists.
3. Provide a clear and descriptive title and description for your PR.
4. Wait for code review. We will merge it once it passes CI checks!

## Code of Conduct

By participating in this project, you are expected to uphold standard community guidelines. Be respectful, constructive, and welcoming.
